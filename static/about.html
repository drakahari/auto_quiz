About DLMS

DLMS (Drakahari's Learning & Management System) is a self-contained quiz and assessment application designed for serious learning and exam preparation. The user is able to toggle between a timed Exam mode and a flexible Study mode, making DLMS suitable for both high-stakes testing and low-pressure studying. Study mode offers instant feedback to facilitate learning. The correct answer/s will turn green when selected. Incorrect answer/s will be highlighted in red. This immediate feedback mechanism enhances the learning experience by allowing users to learn from their mistakes in real-time. Exam mode simulates a formal testing environment with strict time limits and no feedback until the end. There is pause functionality in Exam mode to accommodate real-world testing scenarios. In pause mode the screen will be blurred out and the timer will stop counting down. A Sample Quiz is included to showcase DLMS's features and capabilities, users can explore and understand how to effectively utilize the system for their learning needs. There is also a Dashboard that tracks quiz performance over time, enabling users to monitor their progress and identify areas for improvement.

Users have the option to upload properly formatted question files or to paste their questions directly into the application. When questions are pasted, powerful parsing tools that leverage regular expressions (regex) are available to help transform the content into the required format. These tools are highly effective but require correct and deliberate use of regex patterns to ensure successful data parsing. Lastly, DLMS allows users to upload logos to customize the appearance of their quizzes, adding a personal touch to the learning experience.

Definition:
DLMS is a self-contained, offline-capable assessment and learning system that combines exam-grade correctness with study-grade flexibility, distributed as a single portable application.

Design Philosophy

DLMS prioritizes correctness, reliability, and user ownership. It does not require internet access, user accounts, or external services. DLMS has strong code functionality that allows users to harness the power of regex to properly parse quiz questions and answers. This ensures that users can create complex quizzes that accurately assess knowledge and understanding.
